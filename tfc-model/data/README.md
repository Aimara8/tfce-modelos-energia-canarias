# data

Carpeta para datos locales utilizados en entrenamiento y pruebas. No se deben subir archivos grandes al repositorio.

Se espera colocar aquí las versiones locales (copias) de los datasets procesados desde `tfc-datasets/outputs/`.

Archivos típicos:
- `final_demand_consumption_dataset.csv`
- `final_renewable_generation_dataset.csv`

Consejo: mantener un archivo `.gitignore` para evitar subir CSVs pesados.
